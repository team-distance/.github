## 1. 성과 및 결과
⭐️ 요약 : 우리 프로젝트는 대학교 축제 기간 동안 랜덤 채팅 서비스를 제공하여 누적 가입자수 1600명과 누적 조회수 100만건의 이벤트를 기록했습니다.
  
### a. GA분석결과
GA(Google Analytics)란 웹사이트와 모바일 앱의 사용자 행동 및 성과를 측정하고 분석할 수 있는 웹 분석 도구.<br> 이를 통해 웹사이트나 앱의 트래픽, 사용자 활동, 전환율 등을 모니터링하고 데이터 기반으로 의사결정 가능. <br/>
<br/>

### i. 사용자 증가 및 참여도
<ul>
  <li>축제 기간 동안 사용자가 집중적으로 증가하였음.</li>
  <li>특히 5월 순천향대학교와, 11월 전남대학교 축제 기간에 사용량이 가장 활발했음.</li>
</ul>

### iv. DAU 성과
<ul>
  <li>DAU는 평균 43명, 축제 기간 최고치는 104명을 기록하였음.</li>
  <li>해당 학교 축제 기간에만 유저가 몰리는 한계성을 인지하여 유저가 서비스 내에 오래 머물 수 있는 수단을 찾고자 함.</li>
  <li>이에 축제 페이지를 새롭게 리뉴얼하여 위치 기반으로 내 주변 다른 학교 축제 정보도 검색이 가능하도록 만듦. </li>
</ul>
